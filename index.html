Welcome to Cobalt CSS

Cobalt CSS is a mini CSS toolkit with a few basic styles to get you started. It's designed to be part of a much bigger yet-to-be-announced project, but it seems sensible to release it as soon as possible.

This is very much just an initial release, so please excuse any bugs!
